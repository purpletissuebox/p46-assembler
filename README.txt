# Installation and Usage

1. Install Lark (https://github.com/lark-parser/lark)
2. Install Draftsman (https://github.com/redruin1/factorio-draftsman)
3. Write your program.
4. Assemble: `python assembler.py myProgram`
5. Link: `python linker.py a.out`
6. The provided blueprint string can be used to load the program into the CPU.
